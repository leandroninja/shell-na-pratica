# Shell na Prática

Introdução ao Shell Script com exemplos práticos e progressivos — do básico ao uso de funções e manipulação de arquivos.

## Scripts

| Arquivo | Conteúdo |
|---------|----------|
| `hello.sh` | Primeiro script — olá mundo |
| `variaveis.sh` | Variáveis, tipos e entrada do usuário |
| `condicional.sh` | if/elif/else e estrutura case |
| `lacos.sh` | for, while e iteração sobre arquivos |
| `funcoes.sh` | Declaração e chamada de funções |
| `arquivos.sh` | Criar, ler, verificar e remover arquivos |

## Como executar

```bash
# Dar permissão de execução
chmod +x script.sh

# Executar
./script.sh

# Ou diretamente com bash
bash script.sh
```

## Pré-requisitos

- Linux, macOS ou WSL no Windows
- bash 4.x+
